# Ecoleta - Next Level Week
<img src="https://github.com/gsbenevides2/nlw-ecoleta/raw/master/logo.png" width="100%"/>

O Ecoleta é uma plataforma onde coletores de material para reciclagem podem encontrar pessoas interessadas em entregar seus descartáveis. Esse projeto é resultado da Next Level Week 1 da [Rocketseat](https://github.com/rocketseat)

## Estrutura de pastas
- **server:** backend do projeto, servidor responsável por se conectar ao banco de dados.
- **web:** frontend web do projeto construído em React e React Dom.
